NEVER LOOK LONELY AGAIN! ALWAYS HAVE SOMEONE TO TEXT WITH!

dial the number in the send_sms.py file to be linked with a stranger.
text it up, and when you've had enough, type NEXT to move on to the 
next person, or type PEACE to exit the roulette completely.

Made with twilio api; hosted on google app engine.
